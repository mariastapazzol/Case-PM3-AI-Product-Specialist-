# SuportePro Assistente MVP

Projeto React + Vite pronto para publicar na Vercel ou Netlify.

## Rodar localmente

```bash
npm install
npm run dev
```

## Publicar na Vercel

1. Suba esta pasta para um repositório no GitHub.
2. Acesse a Vercel.
3. Clique em Add New Project.
4. Importe o repositório.
5. Build command: `npm run build`
6. Output directory: `dist`
7. Clique em Deploy.

## Atenção sobre IA real

O arquivo original chama a API da Anthropic diretamente no navegador. Para um protótipo público, o ideal é mover essa chamada para um backend/serverless function antes de usar uma chave real.
